# 01_overview ディレクトリ

## このディレクトリについて

このディレクトリには、AWS基本設計書の全体概要と設計方針が含まれています。

## 含まれるファイル

| ファイル名 | 説明 | 読む順序 |
|----------|------|---------|
| `overview.md` | 設計概要、方針、技術選定のまとめ | 1 |
| `README.md` | このファイル（ディレクトリの説明） | - |

## 読み方

1. **まず `overview.md` を読んでください**
   - プロジェクトの全体像を把握できます
   - 技術選定の理由（ADR）を確認できます
   - 他のディレクトリへのリンクが記載されています

2. **その後、項番順に各ディレクトリを確認してください**
   - 02_account_structure（アカウント構成）
   - 03_network（ネットワーク設計）
   - ... 順番に進めることで理解が深まります

## このディレクトリの役割

- プロジェクトの全体像の提示
- 設計方針の説明
- 技術選定の根拠（ADR）の提示
- 設計書全体のナビゲーション
