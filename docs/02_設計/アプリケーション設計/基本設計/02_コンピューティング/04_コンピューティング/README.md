# 04_compute ディレクトリ

## このディレクトリについて

このディレクトリには、コンピューティング設計（ECS、ALB）が含まれています。

## 含まれるファイル

| ファイル名 | 説明 | 読む順序 |
|----------|------|---------|
| `compute_design.md` | コンピューティング設計書（ECS、ALB） | 1 |
| `ecs_parameters.md` | ECSパラメータシート（具体的な値） | 2 |
| `alb_parameters.md` | ALBパラメータシート | 3 |
| `README.md` | このファイル（ディレクトリの説明） | - |

## 読み方

1. **`compute_design.md`**: コンピューティング設計の全体像を理解
2. **`ecs_parameters.md`**: ECSの具体的なパラメータ値を確認
3. **`alb_parameters.md`**: ALBの設定値を確認

## このディレクトリの役割

- ECS Fargate設計（タスク定義、サービス）
- ALB設計（ターゲットグループ、リスナー）
- Auto Scaling設計
